# PassGuardian V1

Un gestionnaire de mots de passe simple et sécurisé.

## Fonctionnalités

- Stockage de mots de passe.
- Génération de mots de passe.
- Affichage des mots de passe.

## Comment utiliser

1. Exécutez le fichier main.py.
2. L'application s'ouvre.
